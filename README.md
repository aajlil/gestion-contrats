# Gestionnaire de Contrats

Application web développée dans le cadre de mon stage de 2ème année de BUT Informatique.

L'application permet de centraliser et suivre les contrats passés avec 
les fournisseurs de logiciels et de services informatiques de la 
collectivité. Elle gère le cycle de vie complet des contrats, envoie 
des alertes automatiques avant expiration et propose des outils d'analyse et d'export.

## Stack technique

- **Backend** : Node.js, Express
- **Base de données** : PostgreSQL
- **Frontend** : HTML, CSS, JavaScript
- **Architecture** : MVC
- **Bibliothèques** : Chart.js, ExcelJS, PDFKit, Nodemailer, node-cron

## Fonctionnalités

- Authentification avec gestion des rôles (administrateur / utilisateur)
- Création, modification, suppression et consultation des contrats
- Calcul automatique du statut des contrats
- Alertes d'expiration dans l'application et par e-mail
- Historique des modifications
- Export Excel et PDF
- Tableau de bord avec statistiques et graphiques
- Calendrier des échéances
- Gestion des documents associés aux contrats
- Réinitialisation du mot de passe par e-mail
